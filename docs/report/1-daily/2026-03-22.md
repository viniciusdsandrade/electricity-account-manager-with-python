## Decisões técnicas

- **Gemini como parser primário** com fallback regex — garante resiliência sem perder a melhoria de qualidade
- **Módulo único `gemini.py`** — sem camada de abstração desnecessária para um único provider
- **Degradação graciosa** — sem API key, o sistema funciona normalmente com regex + warning (sem insights)
